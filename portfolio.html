  • SFXR
  • Microsoft Visual Studio 2019
Plugins
  • XNA Game Studio
  • Game Framework
Remarks
  • Inspired by the game Asteroids.
  • Game Start sequence utilizes the Arrive steering behavior.
  • Ships fly through space by inertia, following their linear momentum until a force changes it.
  • Ships have normal shots and special attacks.
  • Alpha ship (Red) has a Homing Missile, which applies the Pursuit steering behavior to intercept the Beta ship.
  • Beta ship (Blue) has a Anti-Missile, which applies the Seek steering behavior to intercept the Homing Missile.
  • Asteroids randomly move via Interpose steering behavior, trying to stay in-between both ships.
  • Background star field has 1024 stars with random sizes, and 1/4 of these stars move horizontally to create a space-like effect.
  • Object-oriented approach with Design Patterns like Singleton, State and Strategy.
  • AABB collision detection.
